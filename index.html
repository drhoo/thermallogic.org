Beyond cooling.

Thermal Logic explores heat not merely as waste to move, but as exchange, gradient, pathway, storage, timing, surface behaviour, vibration, resonance, conversion, and possibly addressable thermal packets.

热逻辑研究的不只是“冷却”或“散热”，而是把热理解为交换、梯度、路径、 储存、时序、表面行为、振动、共振、转换，以及未来可能被寻址的热行为单元。

Open mind. Hard floor.

This project welcomes unusual questions, historical clues, and fresh models, but every idea must return to known physics, clear definitions, repeatable observation, and modest claims.

开放心态，坚实底线。欢迎新的问题、历史线索和不同模型，但所有观点都必须回到 已知物理、清晰定义、可重复观察和谨慎结论。

Dynamic Thermal Control

The working frame is Dynamic Thermal Control: a design language for routing, delaying, storing, shaping, selecting, and converting thermal behaviour.

Thermal Logic

Thermal Logic is the component language underneath that frame: source, sink, pathway, gate, delay, buffer, surface, medium, boundary, and exchange.
